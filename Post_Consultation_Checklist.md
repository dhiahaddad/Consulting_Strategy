## Red Flags to Address

If any of these occurred, reflect and plan to improve:

- ⚠️ **Ran significantly over time**
  - Improve time management
  - Set stricter boundaries
  - Better scope setting upfront

- ⚠️ **Client seemed confused at end**
  - Simplify explanations
  - Use more examples
  - Check understanding more frequently
  - Provide better documentation

- ⚠️ **Couldn't answer key questions**
  - Identify knowledge gaps
  - Research for future
  - Build better preparation routine
  - Know when to defer and research

- ⚠️ **Technical difficulties**
  - Better pre-session testing
  - Have backup plans ready
  - Improve setup

- ⚠️ **Scope creep**
  - Better goal setting upfront
  - Practice saying "let's schedule follow-up for that"
  - Clearer communication about time limits
